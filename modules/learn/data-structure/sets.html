Sets

A collection of data that enforces unique data set and implements union, intersection, and complement functions to other sets.

sets = ['a', 'b', 'c']


Insert

Adding a data will append towards the end of the sets if it does not already exist.

sets.insert('b') ⟶ ('b' already exist in sets)
sets.insert('d')
sets[0] = 'a'
sets[1] = 'b'
sets[2] = 'c'
sets[3] = 'd'


Union

The union of two sets contains all the elements contained in either set (or both sets).

sets_a = ['a', 'b', 'c']
sets_b = ['b', 'c', 'd']
union(sets_a, sets_b) ⟶ ['a', 'b', 'c', 'd']


Intersection

The intersection of two sets contains only the elements that are in both sets.

sets_a = ['a', 'b', 'c']
sets_b = ['b', 'c', 'd']
intersection(sets_a, sets_b) ⟶ ['b', 'c']


Complement

The complement of a sets_a contains everything that is not in sets_b.

sets_a = ['a', 'b', 'c']
sets_b = ['b', 'c', 'd']
complement(sets_a, sets_b) ⟶ ['a']
complement(sets_b, sets_a) ⟶ ['d']
