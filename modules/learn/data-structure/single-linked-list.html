Single Linked List

A one-way linear node-to-node data structure which every node holds a reference to the next node.

single_linked_list = node('a') → node('b') → node('c') → node('d')


Search

Since this data structure only knows it's next node, it will have to recursively search the head node to the tail node for the data set.

search(node('a'), 'c')
search(node('a'), 'c') ⟶ (not equal; search next node)
search(node('b'), 'c') ⟶ (not equal; search next node)
search(node('c'), 'c')


Insert

Recursively search the tail node and set the next node to data.

single_linked_list = node('a') → node('b') → node('c') → node('d')
insert(single_linked_list, 'e')
single_linked_list = node('a') → node('b') → node('c') → node('d') → node('e')


Deletion

Upon a deletion of a middle node, relink the previous_node.next_node to the deleted_node.next_node.

single_linked_list = node('a') → node('b') → node('c') → node('d') → node('e')
delete(single_linked_list, 'b')
single_linked_list = node('a') → node('c') → node('d') → node('e')
