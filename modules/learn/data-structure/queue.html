Queue

A list of data that consumes the data via first in, first out (FIFO).

queue = ['a', 'b', 'c']


Enqueue

Adding a data will append towards the end of the queue.

queue.enqueue('d')
queue[0] = 'a'
queue[1] = 'b'
queue[2] = 'c'
queue[3] = 'd'


Dequeue

Removing a data will dequeue the first element of the queue.

queue.dequeue() ⟶ 'a'
queue[0] = 'b'
queue[1] = 'c'
queue[2] = 'd'
