Double Linked List

A two-way linear node-to-node data structure which every node holds a reference to the previous node and next node.

double_linked_list = node('x') ⇄ node('y') ⇄ node('z') ⇄ node('a') ⇄ node('b') ⇄ node('c')


Search

Since this data structure only knows it's previous and next nodes, it will have to recursively search in both direction to each end nodes.

search(node('a'), 'c')
search(node('a'), 'c') ⟶ (not equal; search left side previous node)
search(node('z'), 'c') ⟶ (not equal; search right side next node)
search(node('b'), 'c') ⟶ (not equal; search left side previous node)
search(node('y'), 'c') ⟶ (not equal; search right side next node)
search(node('c'), 'c')


Previous

The left node of the current selected node

previous(node('a')) ⟶ node('z')
previous(node('x')) ⟶ null


Next

The right node of the current selected node

next(node('a')) ⟶ node('b')
next(node('c')) ⟶ null


Insert

Recursively search for the right end node and set the next node to data.

double_linked_list = node('x') ⇄ node('y') ⇄ node('z') ⇄ node('a') ⇄ node('b') ⇄ node('c')
insert(node('x'), 'd') ⟶ node('x') ⇄ node('y') ⇄ node('z') ⇄ node('a') ⇄ node('b') ⇄ node('c') ⇄ node('d')


Deletion

Upon a deletion of a middle node, relink the previous_node.next_node to the deleted_node.next_node.

double_linked_list = node('x') ⇄ node('y') ⇄ node('z') ⇄ node('a') ⇄ node('b') ⇄ node('c')
delete(node('x'), 'z') ⟶ node('x') ⇄ node('y') ⇄ node('a') ⇄ node('b') ⇄ node('c')
