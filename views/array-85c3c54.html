Array

A index-reference collection of similar data fixed in initial size.

array = ['a', 'b', 'c']


Index

The data in array is referenced by incremental indexes starting from 0.

array[0] ⟶ 'a'
array[1] ⟶ 'b'


Insert

Adding a data will append towards the end of the array.

array.insert('d')
array[0] = 'a'
array[1] = 'b'
array[2] = 'c'
array[3] = 'd'


Length

The length is incremented and decremented whenever data is added or removed.

array.length() ⟶ 4
