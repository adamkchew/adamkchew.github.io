Stack

A list of data that consumes the data via last in, first out (LIFO).

stack = ['a', 'b', 'c']


Push

Adding a data will append it towards the end of the stack.

stack.push('d')
stack[0] = 'a'
stack[1] = 'b'
stack[2] = 'c'
stack[3] = 'd'


Pop

Removing a data will pop the last element of the stack.

stack.pop() ⟶ 'd'
stack[0] = 'a'
stack[1] = 'b'
stack[2] = 'c'
